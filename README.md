# Udacity Data Analyst Nanodegree

This repository contains my work for the Udacity Data Analyst Nanodegree program.

## Course Structure

- IntroToDataAnalysis: Foundational concepts and Python data analysis libraries
- DataWrangling: Techniques for cleaning and preparing data
- ExploratoryDataAnalysis: Statistical analysis and visualization
- DataVisualization: Advanced visualization and storytelling
- Capstone: Final project demonstrating all skills

## Technologies Used

- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebooks
- SQL
- Tableau
